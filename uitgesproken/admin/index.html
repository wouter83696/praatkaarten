Verbinden met GitHub

Vul je repository en een Personal Access Token in. Je gegevens worden lokaal opgeslagen.

Classic token (aanbevolen):
  1. Ga naar github.com/settings/tokens/new
  2. Naam + No expiration, vink repo aan (het grote vakje)
  3. Klik Generate token en plak hier
Fine-grained token (alternatief):
  1. Ga naar github.com/settings/tokens?type=beta
  2. Klik Generate new token
  3. Kies bij Repository access: Only select repositories ŌåÆ kies je repo
  4. Klik Repository permissions en zet Contents op Read and write
  5. Genereer en plak hier
