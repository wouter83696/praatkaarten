Verbinden met GitHub

Vul je repository en een Personal Access Token in om sets te beheren.

Token aanmaken (1 min):
  1. Ga naar github.com/settings/tokens/new
  2. Naam invullen, kies No expiration
  3. Vink repo aan (alle repo-rechten)
  4. Klik Generate token en plak hier
